# Ebrahim Ali Portfolio

Modern React portfolio built with Vite, Tailwind CSS, DaisyUI, Framer Motion, and EmailJS.

## Live

https://web-ebrahim-portfolio.vercel.app/

## Features

- Neon-themed UI and animated sections
- Project gallery with filter and details pages
- About timeline and skills overview
- Contact form powered by EmailJS
- Responsive layout for mobile and desktop

## Tech Stack

- React
- Vite
- Tailwind CSS
- DaisyUI
- Framer Motion
- React Router
- React Hook Form
- EmailJS

## Environment Variables

Create a `.env` file:

```bash
VITE_EMAILJS_SERVICE_ID=your_service_id
VITE_EMAILJS_TEMPLATE_ID=your_template_id
VITE_EMAILJS_PUBLIC_KEY=your_public_key
```

## Development

```bash
npm run dev
```

## Build

npm run build
